# Labs Nutrition — Sistema de Seguimiento de Clientes

Sistema web para importar ventas de Tienda Nube, ver el panel de clientes y reactivar los que no compran hace tiempo.

## Cómo usar

1. Exportá el CSV de ventas desde **Tienda Nube → Ventas → Exportar**
2. Abrí la web y arrastrá el archivo en la sección **Importar**
3. El panel y los mensajes se actualizan solos

## Cómo subir a Vercel

1. Subí esta carpeta a GitHub
2. Entrá a vercel.com → New Project
3. Conectá el repositorio
4. Hacé clic en Deploy

¡Listo! Tenés tu sistema online.
